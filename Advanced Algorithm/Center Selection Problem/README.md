### 配置如下

+ site数目：1000
+ site坐标范围：(0,100000), double类型, 均为随机生成
+ k值变化范围[5,200], 步长为5
+ 对于每组实验重复200次，统计每个算法赢的册数

### 算法

1. 选择的起始节点到其他所有节点的距离之和最小
2. 选择的起始节点到其他所有节点的距离之和最大
3. 随机选择起始节点
